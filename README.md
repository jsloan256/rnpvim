#rnpvim: Rafael Pereira's vim config files


## Installation

Get the rnpvim code:
git clone https://https://github.com/rafaelnp/rnpvim.git rnpvim

`ln -s your-path/rnpvim/.vim ~/.vim`

`ln -s your-path/rnpvim/.vimrc ~/.vimrc`

and execute:

`vim +BundleInstall`
